# NewsGT

> The news, without the noise.

NewsGT is an open-source geopolitical news analysis platform built on the belief that most people know *what* happened — but rarely *why*.

---

## What it does

NewsGT ingests news coverage from across the global media spectrum and surfaces what actually happened, who is involved, and what is driving their behavior — stripped of framing, spin, and agenda.

It is not a news aggregator. It does not show you more headlines. It shows you less — but more accurately.

---

## Principles

- **No topic filtering.** The algorithm is the integrity, not the content selection.
- **No hidden actors.** Analysis is grounded in documented, publicly verifiable facts.
- **Confidence over certainty.** Every analytical claim is labeled by how well it is supported.
- **Objectivity over validation.** The goal is accuracy, not comfort.

## Tech Stack

- **Pipeline:** Python 3.11
- **Database:** PostgreSQL via Supabase
- **News ingestion:** GDELT DOC 2.0 API
- **Frontend:** Next.js
- **Deployment:** Vercel / Railway
